🔊 MIR Feature Extraction

Extract tempo, key, timbre, emotional valence, energy from audio. Pure signal processing — a legitimate CS problem.

Use extracted features to find or generate music with target properties — useful for therapy, focus, exercise, scoring.

Music therapists do analysis manually. A tool that helps them — the professionals — solves a real problem.
